# connect — Terminal interactiva

## Descripción
Abre una terminal SSH interactiva con el servidor activo.

## Sintaxis
```bash
sshcli connect
```

## Ejemplos
```bash
# Conectar al servidor activo
sshcli connect
```

## Notas
- Requiere servidor activo (`sshcli server use`)
- Sesión interactiva con TTY
- Útil para comandos que requieren interacción
- Ctrl+D para salir
