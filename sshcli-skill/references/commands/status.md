# status — Estado de conexión

## Descripción
Verifica el estado y salud de la conexión con el servidor activo.

## Sintaxis
```bash
sshcli status
```

## Ejemplos
```bash
# Ver estado de conexión
sshcli status
```

## Notas
- Muestra si el servidor está reachable
- Indica tiempo de respuesta
- Útil para diagnosticar problemas de conexión
